GEANT4 Simulation for an ionization chamber to be used with (d, n) reactions using VANDLE (Versatile Array of Neutron Detectors at Low Energy). The current configuration includes grids for energy loss in a gas and a scintillator to be used for timing and energy.\hypertarget{md_sourse_README_autotoc_md1}{}\doxysection{Getting Started}\label{md_sourse_README_autotoc_md1}

\hypertarget{md_sourse_README_autotoc_md3}{}\doxysubsection{Build and Run}\label{md_sourse_README_autotoc_md3}
To build and run the simulation with the above packages installed, create and move to a build directory


\begin{DoxyCode}{0}
\DoxyCodeLine{mkdir build \&\& cd build}

\end{DoxyCode}


Use CMake to then build and make the simulation executable


\begin{DoxyCode}{0}
\DoxyCodeLine{cmake .. \&\& make}

\end{DoxyCode}


To run the simulation, use the created executable with a configuration file (a sample is provided)


\begin{DoxyCode}{0}
\DoxyCodeLine{./mmSim config.json -\/t N}

\end{DoxyCode}
 where N is the number of threads to run for multi-\/threading.\hypertarget{md_sourse_README_autotoc_md4}{}\doxysubsection{Configuration Files}\label{md_sourse_README_autotoc_md4}
There are two configuration files provided needed to run the simulation\+: run.\+mac and config.\+json.\hypertarget{md_sourse_README_autotoc_md5}{}\doxysubsubsection{run.\+mac}\label{md_sourse_README_autotoc_md5}
The current example run.\+mac file looks like this\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{/run/printProgress 1000}
\DoxyCodeLine{/run/beamOn 50000}

\end{DoxyCode}


This will run 50,000 events and print the progress every 1,000 events.\hypertarget{md_sourse_README_autotoc_md6}{}\doxysubsubsection{config.\+json}\label{md_sourse_README_autotoc_md6}
The config.\+json is the main file you will be editing. You can change some (not all) of the geometry, gas, reaction products and resolution parameters.

An explanation of each configuration parameter is described below\+:


\begin{DoxyItemize}
\item \char`\"{}interactive\char`\"{}\+: When this is set to true, it will load up a GUI to visualize the detector setup and geometry. This can be only set to true or false.
\item \char`\"{}macro\+Name\char`\"{}\+: This is macro file to run automatically when \char`\"{}interactive\char`\"{} is false. It\textquotesingle{}s parameters are described above.
\item \char`\"{}gas\+Type\char`\"{}\+: This is the name of the gas to use in the ionization chamber. Currently there are four gases to choose from\+: “\+P10” (Argon 90\% CH4 10\%), “\+CH4”, “\+CO2” and “\+CF4”. Other gases can be added by editing the \mbox{\hyperlink{classMMDetectorConstruction}{MMDetector\+Construction}} files.
\item \char`\"{}gas\+Temperature\char`\"{}\+: This is temperature of the above gas in Kelvin.
\item \char`\"{}gas\+Pressure\char`\"{}\+: This is the pressure of the above gas in Torr.
\item \char`\"{}number\+Grids\char`\"{}\+: This is the total number of cathode-\/anode-\/cathode grids to use in the setup.
\item \char`\"{}grid\+Size\char`\"{}\+: This is the total distance from cathode to cathode. Cathode to anode distance is half of this value. This can be in mm or inches and a parameter below will describe this.
\item \char`\"{}grid\+Radius\char`\"{}\+: This is the radius of the grids assuming they are circular. This can be in mm or inches and a parameter below will describe this.
\item \char`\"{}grid\+Distance\char`\"{}\+: This is the distance from the window (entrance of the ionization chamber) to the middle of the IC grids. This can be in mm or inches and a parameter below (\char`\"{}use\+Inches\char`\"{}) will describe this.
\item \char`\"{}scint\+Distance\char`\"{}\+: This is the distance from the window (entrance of the ionization chamber) to the scintillator. This can be in mm or inches and a parameter below (\char`\"{}use\+Inches\char`\"{}) will describe this.
\item \char`\"{}use\+Inches\char`\"{}\+: When this is set to true, \char`\"{}grid\+Distance\char`\"{}, \char`\"{}grid\+Radius\char`\"{} and \char`\"{}distance\+Scint\char`\"{} will be in inches. When it is false, they will be in mm.
\item \char`\"{}scint\+Resolution\char`\"{}\+: This is the resolution of the scintillator resolution (1 is 100\%).
\item \char`\"{}extra\+Grid\+Resolution\char`\"{}\+: Just using the fano\+Factor will still under estimate the errors of the energy in the grids. This is an extra resolution on top of that (1 is 100\%).
\end{DoxyItemize}\hypertarget{md_sourse_README_autotoc_md7}{}\doxysubsection{Output Files}\label{md_sourse_README_autotoc_md7}
